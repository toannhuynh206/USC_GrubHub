Deployment Information :

I am currently working on deploying this project live to google so all users can use, but for right now in order to run this website you must be using eclipse and tomcat 9.

Steps for Deployment

1. git pull from the resposityry and compress the file into a zip
2. import the zip file into eclipse ,  doing the import method that imports an existing project
3. run the sql script within sql with an account with username root, root
4. run the logout.jsp using a tomcat 9 server
